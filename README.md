# Quiz-App
Quiz-App using React JS.

Quiz App
A modern, interactive quiz application built with React, featuring a sleek UI. The app dynamically renders questions, keeps track of the user's score, and displays the result at the end.

Features
Dynamic question rendering.
Reset functionality for retaking the quiz.
User-friendly interface.
Responsive design for desktop and mobile.

Technologies Used
React: Front-end framework for building the app.
React Router: For managing navigation between pages (Quiz and Result).
Tailwind CSS: For styling and ensuring responsive design.
Context API: For state management across components.
